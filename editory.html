Editory kódu

Úvod do editorů kódu

Nejjednodušeji můžete začít psát kód ve vestavěném editoru textu ve svém operačním systému windows linux nebo mac a systémech linux. Většina z vás s němi bude už seznámena. Budete je tedy moct používat hned bez žádného učení. Avšak nepodporují například otevření projektu práci se složkami aatd.

Pokročilejší editory

Používaním pokročilejších editorů jako je třeba Atom nebo VS code, Atom je jednoduší na ovládání, ale VS code má český překlad. Pokud chcete začít a umíte anglicky, začněte Atomem. Pokud chcete víc pokročilejší editor a/nebo chcete češtinu, šáhněte po VS code.

IDE

IDE nejsou jenom editory kódu, umožňují vám kód i testovat a tím pádem i vidět chyby bez hledání. Některé jsou specializované na jeden programovací jazyk, některé ne. Například Visual Studio umožňuje doinstalovat rozšíření pro další jazyky, ale například IDE od Jet Brains jsou každé specializované pro jeden programovací jazyk. Stejně jako v minulé kapitole o editorech Visual Studio je v Češtině a IDE od Jet Brains nejsou.
